## Description

A Node js web application that collects sign-up form data and sends it directly to an audience list on MailChimp via MailChimp's API.

## Features

- Front-end Form Validation
- Asynchronous Programming
- API Integration

## Tech Used

- Node
- Express
- EJS
- Axios
- HTML & CSS
